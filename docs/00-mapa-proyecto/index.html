Estructura del Repo

Aplicacion

  • app/Http/Controllers: controladores web y API.
  • app/Services: reglas de negocio principales.
  • app/Models: modelos Eloquent.
  • app/Http/Requests: validaciones.
  • app/Console/Commands: comandos artisan propios.

Interfaz

  • resources/views: Blade web.
  • resources/views/layouts: layouts base.
  • resources/views/components/ds: componentes del design system.
  • resources/css/app.css: tokens y estilos globales.

Datos y ejecucion

  • database/migrations: estructura de BD.
  • database/seeders: datos iniciales.
  • database/dump.sql: dump versionado.
  • routes/web.php y routes/api.php: entrada web/API.

Documentacion

  • docs/00-estado: estado vigente.
  • docs/02-contratos: contratos de negocio.
  • docs/03-diseno-ui: design system.
  • docs/99-archivo: historico.
